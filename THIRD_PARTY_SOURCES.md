# Third-party Sources

This project aggregates Claude Code skills from third-party repositories. The table below records the current sources and attribution notes.

| Source | Current count | Attribution / license note |
| --- | ---: | --- |
| [ComposioHQ/awesome-claude-skills](https://github.com/ComposioHQ/awesome-claude-skills) | 843 | Third-party skill content. Several skill folders include their own `LICENSE.txt`; many inspected license files are Apache-2.0. Check upstream for exact terms. |
| [anthropics/skills](https://github.com/anthropics/skills) | 19 | Third-party skill content. Several skill folders include their own `LICENSE.txt`. Check upstream for exact terms. |
| [JimLiu/baoyu-skills](https://github.com/JimLiu/baoyu-skills) | 23 | Third-party skill content. No unified top-level license was detected during inspection; check upstream before reuse beyond indexing/personal installation. |
| [stellarlinkco/myclaude](https://github.com/stellarlinkco/myclaude) | 13 | Third-party skill content. The upstream repository declares AGPL-3.0. |

Counts are generated from `data/skills.json` and may change after sync.

## How attribution appears in the UI

Each skill detail panel includes:

- repository link
- repository path
- original description
- license note

## Removal or correction

If an upstream author wants attribution changed or mirrored content removed, please open an issue in this repository.
